# Multi-Project ML Collection

This repository contains five distinct machine learning projects covering Computer Vision and Natural Language Processing.

## Projects

1. **1-Fashion_MNIST**: A Convolutional Neural Network (CNN) trained to classify clothing items from the Fashion MNIST dataset.
2. **2-Emotion_NLP**: An NLP project that uses LSTM to classify text comments into six different emotions (joy, sadness, anger, fear, love, surprise).
3. **3-Object_Recognition**: A computer vision project utilizing transfer learning with the ResNet50 architecture to classify images from the CIFAR-10 dataset.
4. **4-Resume_NLP**: An NLP application that uses TF-IDF vectorization and an SVM classifier to categorize resumes into various job roles.
5. **5-Face_Mask_Detection**: A CNN-based project that detects whether individuals in images are wearing face masks.

## Setup

To run these projects locally, ensure you have Python 3.8+ installed and install the dependencies:

```bash
pip install -r requirements.txt
```

Each folder contains a `notebook.ipynb` that can be run in Jupyter Notebook, JupyterLab, or VS Code.

## Dependencies

The projects rely on the following key libraries:
- TensorFlow / Keras
- Scikit-learn
- Pandas
- Numpy
- Matplotlib / Seaborn
- OpenCV (cv2)
- NLTK
- WordCloud
